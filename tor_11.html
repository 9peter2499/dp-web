ข้อเสนอแนะคณะกรรมการ :
คณะกรรมการเสนอให้มีหน้าจอสำหรับการใช้งานโดย Admin ของ PCS เพื่อแสดงให้เห็นสิทธิ์ของ PCS Admin ที่มีสถานะ Active / Inactive รวมถึงการกำหนด Super User PCS Admin ได้

อ้างอิง PDPA ในหน้า Create User ให้ซ่อน และแสดงผลเป็น ******* หรือ xxxxxxx ข้อมูลส่วนบุคคล เช่น อีเมล์ เบอร์โทร เลขที่บัตรประชาชน ซึ่งผู้ที่จะเห็นข้อมูลนี้ ต้องเป็น Super User Admin เท่านั้น
วันที่ : 14 May 2025
รายละเอียดการแก้ไข :
จากข้อเสนอแนะของคณะกรรมการในเรื่องดังกล่าวนี้ ทีมได้มีการเพิ่ม Function เข้าไปในระบบ Setup เพิ่มเพื่อให้
1. กรณีผู้ที่ไม่มีสิทธิ์ในการเห็นข้อมูล ข้อมูลที่เป็นข้อมูลอ่อนไหว เช่น ชื่อ นามสกุล หมายเลขบัตรประชาชน เป็นต้นจะถูกเข้ารหัสไว้ ซึ่งการเข้ารหัสดังกล่าว สามารถที่จะกำหนดรูปแบบได้เอง โดยสามารถเข้าถึงข้อมูล Setup นี้ ตาม hyperlink "INTELLIGENCE PARAMETER->PDPA Configuration"
2. การ Setup ดังกล่าวสามารถที่จะเปิด-ปิดการแสดงได้ตามนโยบายขององค์กรกำหนด
